Project Terbaru

Selama saya belajar programming hingga saat ini saya sudah berhasil membuat beberapa website dari awal hingga yang terbaru. Berikut adalah project yang saya miliki:

Pendaftaran Online Pasien

Project pertama saya adalah membangun Pendafataran Online Pasien Rawat Jalan berbasis Android di Klinik Pratama Ary Farma Jaten Karanganyar dan terintegrasi ke website yang merupakan project untuk Tugas Akhir saya semasa kuliah di Universitas Duta Bangsa Surakarta

Rekam Medis Elektronik

Project kedua saya merupakan hasil dari pengembangan saya yang mencoba untuk membangun Sistem Rekam Medis Elektronik Berbasis Web di Klinik Pratama Ary Farma. Project ini saya bangun untuk membuat pelayanan di suatu fasyankes terutama Klinik sudah terkomputerisasi untuk pelayanan yang lebih cepat dan efisien

Sistem Informasi Manajemen Rumah Sakit

Project ini saya buat dengan framework Codeigiter 3 dan bootstrap template admin-LTE yang merupakan pengembangan dari sistem yang saya buat sebelumnya. Sistem ini dirancang cukup kompleks dengan user multi-login dan database kompleks untuk keperluan Pelayanan di suatu Rumah Sakit. Hingga saat ini sistem ini masih terus dikembangkan untuk menjadi lebih baik.
